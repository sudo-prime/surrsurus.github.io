  • Why 'Free Software'

  • fsf.org/about/what-is-free-software

There are many benefits to free software. It allows for the user to run the code how they wish, study it, make changes to it, and redistribute those changes as a new program. Free software creates valuable learning tools for other programmers and people who want to program. In my opinion, it also allows for better quality programs through scrutiny. My code has improved greatly by allowing others to view it on GitHub. GitHub is a great place for free software as users can raise issues the program has, directly view its source code, suggest direct changes to the code, and create their own variations of projects.
