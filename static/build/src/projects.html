  • Deercode

  • View the source

Deercode is a small scripting language with a focus on being LISP-like and functional. It is currently only implemented in Python, but it boasts many features, such as inheriting all of Python's math module functions as deercode functions. It is still a work in progress, but it comes with many example programs and a small math library, all in deercode.

  • EdgeQuest

  • View the source

EdgeQuest is a humorous roguelike game made in Python with libtcod. It features ranged combat, magic, armor and inventory, potions, varying AI, and a dog companion amongst other features. Battle your way through 12 dungeon floors to look for the ancient fedora artifact.

  • Edgequest Dungeon Generator Demo

  • View the source

This is a proof of concept dungeon generator for Mt. Autumn's future game project, Edgequest. It features infinite travel on the horizontal plane and z-axis, random structures, and basic AI. It uses javascript, meaning you can run this in your browser.

  • Jabberwock

  • View the source

Jabberwock is a small ncurses based roguelike made in Ruby. It has 3 different enemy types, a point system, and a life system. I made this to see if a roguelike could be developed in ruby in a short amount of time, and the answer is yes. The play time of each game is around 2-10 minutes.

  • Enaml

  • View the source

Enaml is not a markup language, but a markup language to html converter for a custom markup language. It's all done in Scala, which compiles to java bytecode and runs in the JVM.
